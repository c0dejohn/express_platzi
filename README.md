# Que es express

es un framework para crea aplicaciones we, apis, web services

### caracteristicas
 1. minimalista
 2. Template engines
 3. Routing
 4. Middlewares
 5. Plugins(as middlewares)

### creando tu primer servidor

instalamos la dependecia de express
`npm i -S express`  
-S grabar express en el directorio del proyecto
